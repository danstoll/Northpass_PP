# Database Reports

The Reports section provides on-demand analytics about partners, users, and certifications.

## Available Reports

### Partner Analytics

| Report | Description |
| ------ | ----------- |
| **Overview Dashboard** | High-level statistics and KPIs |
| **Partner Leaderboard** | Top partners ranked by NPCU |
| **Certification Gaps** | Partners below their tier threshold |
| **Partners Without Groups** | Partners not linked to LMS groups |

### User Reports

| Report | Description |
| ------ | ----------- |
| **User Certifications** | All users with their certifications |
| **Contacts Not in LMS** | CRM contacts without LMS accounts |
| **Inactive Users** | Users with no recent activity |

### Course & Activity

| Report | Description |
| ------ | ----------- |
| **Popular Courses** | Most completed courses |
| **Recent Activity** | Latest enrollments and completions |
| **Expiring Certifications** | Certs expiring in next 90 days |

## Running a Report

1. Click on the report card
2. Wait for data to load
3. Review results in the table
4. Use filters to narrow down results
5. Export data if needed

## Report Features

### Pagination

Large reports are paginated at 1000 records per page. Use the page navigation to browse through results.

### Filtering

Most reports support filters:

- **Region** - Filter by geographic region (AMER, EMEA, APAC)
- **Tier** - Filter by partner tier
- **Owner** - Filter by account owner (PAM)

### Sorting

Click column headers to sort by that column. Click again to reverse the sort order.

### Export

Export report data to CSV or Excel for further analysis in spreadsheet software.

## Report Descriptions

### Overview Dashboard

Shows key metrics at a glance:

- Total active partners
- Total NPCU across all partners
- Users with certifications
- Average certifications per partner

### Partner Leaderboard

Ranks partners by total NPCU. Useful for:

- Identifying top performers
- Finding partners to highlight in communications
- Tracking partner program success

### Certification Gaps

Shows partners whose NPCU falls below their tier requirement:

- Premier tier needs 20 NPCU
- Select tier needs 10 NPCU
- Registered tier needs 5 NPCU

Action: Reach out to these partners to encourage certification.

### Contacts Not in LMS

Lists CRM contacts who don't have matching LMS accounts. These contacts may need:

- Invitation to register for the LMS
- Help with account creation
- Follow-up on training requirements

### Expiring Certifications

Shows certifications expiring within 90 days. Proactive notification helps partners maintain their status.

## Scheduled Reports

Currently, all reports are on-demand. Contact your administrator if you need scheduled email reports.
