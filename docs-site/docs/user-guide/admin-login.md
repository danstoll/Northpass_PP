# Admin Dashboard Login

## Access URL

Navigate to: [https://ptrlrndb.prod.ntxgallery.com/admin](https://ptrlrndb.prod.ntxgallery.com/admin)

## Login Process

1. Enter the admin password when prompted
2. Click **Login**
3. Your session will be stored in your browser

:::info
Contact your system administrator if you don't have the password.
:::

## Session Duration

Your login is stored for the browser session. You'll be logged out when you:

- Close the browser completely
- Click **Logout** in the navigation sidebar
- Clear browser data

## Navigation

After login, you'll see the admin sidebar with these sections:

### Main Navigation

| Menu Item | Description |
| --------- | ----------- |
| **Dashboard** | Overview statistics and quick actions |
| **LMS Sync** | Data synchronization controls and history |
| **Reports** | Generate on-demand reports |
| **User Management** | Manage users, groups, and orphan discovery |
| **Data Management** | Browse and clean partner data |
| **Owner Report** | Partner URLs for distribution to PAMs |

### Quick Tips

- The sidebar can be collapsed by clicking the menu icon
- Current page is highlighted in the sidebar
- Use the Portal link in the top-right to return to the main site

## Troubleshooting

**Can't login?**

- Verify you're using the correct password
- Try clearing browser cache and refreshing
- Contact your administrator

**Session expired?**

- Simply re-enter the password
- Check if you closed the browser or cleared cookies
